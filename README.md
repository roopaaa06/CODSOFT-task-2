# 📊 Task‑2: Student Grade Calculator

This project is part of my **Java Programming Internship at CodSoft**.  
It’s a simple application that calculates a student’s total marks, average percentage, and assigns a grade based on performance.

---

## 📌 Features
- Accepts marks input for multiple subjects (out of 100).
- Calculates **total marks**.
- Computes **average percentage**.
- Assigns grades based on percentage ranges.
- Displays results clearly to the user.

---

## 🛠️ Concepts Used
- Arrays for storing subject marks.
- Loops for iterating through inputs.
- Conditional statements (`if‑else`) for grade assignment.
- User input handling (`Scanner`).

---

## 🚀 How to Run
1. Save the file as `StudentGradeCalculator.java`.
2. Compile the program:
   ```bash
   javac StudentGradeCalculator.java
